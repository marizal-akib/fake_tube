Discuss the scope of var, let, and const.

==>var has function-level scope and is hoisted.
==>let and const have block-level scope and are not hoisted.
==>Use let and const for better scoping and to avoid hoisting-related issues in modern JavaScript.

Tell us the use cases of null and undefined.

==>Use null to represent an intentional absence of any object value.
==>Use undefined to represent an uninitialized or missing value, often the default value of uninitialized variables.

What do you mean by REST API?

In short, a REST API (Representational State Transfer Application Programming Interface) is a set of rules and conventions for building and interacting with web services. It uses standard HTTP methods (GET, POST, PUT, DELETE) to perform operations on resources, which are represented as URLs. REST APIs are designed to be stateless and scalable, making them a popular choice for building web services that can be easily accessed and understood by various clients and platforms.
